### Laboratory works by student Liakh Artemii IS-21
### Variant 18

#### Clone the project:
```
git clone https://github.com/ArtemLyah/java-labs
```
#### Checkout to a certain lab branch
```
git checkout lab3
```
#### Build the project:
```
javac -d out $(find . -name "*.java") 
```

#### Run java project
```
java -cp out labs.Main 
```
